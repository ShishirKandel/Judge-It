# Audio Assets

This folder should contain the audio files for background music and sound effects.

## Required Files

Download free royalty-free audio from sites like:
- [Pixabay](https://pixabay.com/sound-effects/)
- [Freesound](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

### Background Music
- `background_music.mp3` - Chill, looping background music (30-60 seconds loop)
  - Search: "chill lofi background loop" or "ambient game music"
  - Should be calm and non-distracting

### Sound Effects
- `swipe_whoosh.mp3` - Short whoosh sound for card swipe (~0.3s)
  - Search: "whoosh" or "swipe sound effect"
  
- `vote_nta.mp3` - Positive ding/success sound for NTA vote (~0.5s)
  - Search: "success ding" or "positive notification"
  
- `vote_yta.mp3` - Neutral/buzzer sound for YTA vote (~0.5s)
  - Search: "buzz" or "wrong answer"
  
- `badge_unlock.mp3` - Celebration/achievement sound (~1s)
  - Search: "achievement unlock" or "level up"

## File Format
- Format: MP3 or OGG
- Sample rate: 44.1kHz
- Keep file sizes small (<500KB each)

## License
Make sure all audio is Creative Commons Zero (CC0) or royalty-free for commercial use.
